# Epic Manhunt

Epic Manhunt is a Minecraft Manhunt plugin.

Its Menu allows for a quick and straightforward configuration, which allows you to play in less than a minute.

## How to set up
- Put the plugin in the 'plugins' folder
- Start your server
- Run the `/manhunt` command to configure and start the manhunt.
- The GUI opend with `/manhunt` is entirely straightforward.

## Configurable options :
- Enable / Disable keepInventory for Hunters
- Enable / Disable glowing effect for speedrunners
- Run the manhunt in the default world or in a newly, automaticly generated world by the plugin

## Upcoming features
- Config file so you don't have to configure back the plugin each restart

## Command
- `/manhunt` - Open the Epic Manhunt Gui

## Permission
- `epicmanhunt.admin` - Permission for the `/manhunt` command
